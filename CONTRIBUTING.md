# Contributing Guidelines

All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

## Definitions

* **Bug Reports:** Notifications of a problem, error, or failure in the script.
* **Bug Fixes:** Submitting code that resolves an existing bug report.
* **Documentation Improvements:** Updating the README or comments in the code to make them clearer.
* **Enhancements:** Adding new features, such as supporting compound interest or adding more user prompts.
* **Ideas:** Suggestions for future updates or changes to the project workflow.
* **All Contributions:** Any other helpful actions, such as code reviews or testing the script in different environments.

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Submit a Pull Request for review.
